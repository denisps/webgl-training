# webgl-training

WebGL2 training and inference framework for small dense networks and transformer blocks, built entirely with render-to-texture GPU shaders and served with Bun.

## Docs
- [architecture.md](./architecture.md)
- [agent.md](./agent.md)

## Quick start
```bash
bun start
```
Open http://localhost:3000.
